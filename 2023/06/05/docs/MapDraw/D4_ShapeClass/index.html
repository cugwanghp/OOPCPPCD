2. 类的接口设计

分析程序的需求，需要Shape类提供什么接口呢？有无统一的接口呢？如有统一的接口，则需要添加为基类的虚函数。

可能的接口包括：

  • Draw - 绘制图形
  • Area - 计算面积
  • Perimeter - 周长
  • read - 从文件读
  • write - 写入文件
  • Move - 移动坐标
  • …
